## Smile Foundation (Website)

Static website starter (HTML/CSS/JS).

### Structure

- `index.html` — Home
- `pages/` — Additional pages
  - `about.html`
  - `programs.html`
  - `get-involved.html`
  - `contact.html`
- `css/style.css` — Global styles
- `js/main.js` — Shared interactions (mobile nav, smooth scroll)
- `assets/`
  - `images/` — Images (placeholders for now)
  - `icons/` — Favicon / icons

### Run

Open `Smile_Foundation/index.html` in a browser (or use VS Code/Cursor “Live Server” if you have it).
